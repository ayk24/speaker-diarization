# Speaker Diarization
## Overview
This is a simple speaker diarization script using `pyannote.audio`.

## Requirements
- Python 3.9+
- poetry 1.6 or later

## Usage
1. Install dependencies
    ```shell
    poetry shell
    poetry install
    ```

2. Run the script
    ```shell
    poetry run python main.py
    ```
    or
    ```shell
    make run
    ```
